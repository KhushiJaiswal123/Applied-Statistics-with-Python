Installing Python

To install Python:

  • Download from the official website: python.org
  • Recommended version: 3.10 or higher
  • For Mac: Consider using Homebrew
  • For Windows: Use the Microsoft Store or installer

Installing VS Code

Steps to install VS Code:

  • Download from: code.visualstudio.com
  • Recommended extensions:
    • Python
    • Quarto
    • GitLens
    • Jupyter

Using GitHub

To get started with GitHub:

  • Sign up at github.com
  • Learn basic commands:
    • git clone
    • git add
    • git commit
    • git push
  • Try GitHub Desktop if you’re not comfortable with the command line interface (CLI)
